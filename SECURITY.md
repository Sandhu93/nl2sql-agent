# Security Policy

## Supported Versions

This project is currently under active development. Security updates are provided for the latest version on the `main` branch.

| Version | Supported |
| ------- | --------- |
| Latest (`main`) | Yes |
| Older versions | No |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public GitHub issue.

Instead, report it privately by emailing: **your-email@example.com**

Please include:
- a clear description of the issue
- steps to reproduce it
- the potential impact
- any suggested fix, if available

You can expect an initial acknowledgment within **3 to 5 business days**.

After reviewing the report, I will:
- confirm whether the issue is valid
- share the planned remediation approach if accepted
- notify you when a fix has been released

If the report is not accepted as a security issue, I will let you know why.

Thank you for helping improve the security of this project.
